Post Danmark - Pickup Locator Module

Commerce Shipping module to enable your customers to choose their own pickup place for thier package.
Google Maps is used during the Commerce Checkout process to choose the pickup place.

Installation
============
- Create a Post Danmark Consumer API at http://www.postdanmark.dk/da/Logistik/netbutikker/login/Sider/Opret.aspx
- Extract "pickup_locator" to /site/all/modules
- Activate the module in /admin/modules.
- Search for "Postnord Pickup Locator (DK)" and activate it
- Click configure and type in a valid Post Denmark API-code and shipping price.
